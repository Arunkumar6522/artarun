CMS stands for Content Management System. A CMS allows content to be edited or added through editors and other simple fields. CMS systems are useful as they allow people with little to no online experience to still be able to input, create and change their website content.

Craft CMS is a content management system (CMS for short) that allows you to edit your website's/app’s content whenever you wish. It’s user-friendly and comes with a responsive control panel so you can edit content whenever, wherever. For developers, it allows full flexibility to allow any idea or project to be editable and scale whilst being secure. It has big features such as multisite, localisation (languages) and a large plugin store, as well as allowing full flexibility over your content with content blocks and custom fields. We wrote an article about why we do not use WordPress.

Even though you might not have heard of it yet, Craft CMS is used by global brands such as Netflix, Ikea, Sonos, Wrangler and Ideo. As well as being chosen as the CMS for the W3C, the people responsible for bringing standardisation to the web by the creator of the web, Tim Berners-Lee.

Craft CMS is built on top of Yii, which is a PHP and MySQL framework. This allows CMS features and security patches to be rolled out quicker than being built on no framework at all. For making your site look the part, it uses Twig to allow us to rapidly develop features for your project. On top of this, it can be integrated easily with modern static sites as it comes with GraphQL natively, allowing it to run as a Headless CMS.

Craft CMS is an open source CMS which means as well as it being commercially developed by Pixel & Tonic (The Craft CMS founders) it allows the community the ability to improve the CMS themselves. Enabling features and security patches to be fixed quicker than if it was being privately developed.
